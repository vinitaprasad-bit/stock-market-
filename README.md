\# Market Share Analysis System



\## Project Summary



Market Share Analysis System is a Python-based data analysis and visualization project developed using \*\*Pandas\*\*, \*\*NumPy\*\*, and \*\*Matplotlib\*\* libraries. The system provides a menu-driven interface that allows users to import market share data from a CSV file, perform various data manipulation operations, and visualize the data through different graphs.



The project enables users to:



\* Import and export market share data using CSV files.

\* Display, search, add, and delete records from the DataFrame.

\* Search records based on columns, share types, and row indexes.

\* Find shares with maximum target price and minimum common market price.

\* Display top and bottom records of the dataset.

\* Add new columns and generate descriptive statistics of the data.

\* Visualize data using Line Plots, Bar Graphs, Horizontal Bar Graphs, and Histograms.



The system follows a menu-driven approach, making it user-friendly and suitable for beginners learning \*\*Data Analysis, Data Manipulation, and Data Visualization in Python\*\*. The project demonstrates practical implementation of DataFrames, file handling, aggregation functions, and graphical representation of financial market data.



\## Technologies Used



\* Python

\* Pandas

\* NumPy

\* Matplotlib

\* CSV File Handling



\## Objective



The main objective of this project is to analyze market share data efficiently, perform data manipulation operations, and generate meaningful visual insights for better understanding of market trends and share performance.
